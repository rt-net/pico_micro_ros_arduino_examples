# micro-ROS Arduino examples for Pi:Co Classic 3

Pi:Co Classic 3用のmicro-ROS Arduino サンプル集です

## スケッチファイルの自動整形について

ソースコードのレイアウトを整えるため、各スケッチファイルにはArduino IDEの自動整形を適用しています。
自動整形のルールは[.clang-format](.clang-format) ファイルを参照してください。
